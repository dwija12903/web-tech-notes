Tailwind CSS Notes

Variants

  • Media Query Variants:
    • Adjust styles based on screen size: sm:text-md md:text-lg
    • Breakpoints: sm, md, lg, xl
  • State Variants:
    • Examples: hover:bg-blue-500, disabled:opacity-50
    • Other states: active, focus, visited
  • Other Variants:
    • Examples: first:mt-0, last:mb-0, odd:bg-gray-100, even:bg-gray-200
    • Dark mode: dark:bg-gray-800, dark:text-white

Typography

Fonts: Use font-sans, font-serif, font-mono.

Font Sizes:

  • text-xs
  • text-sm
  • text-base
  • text-lg
  • text-xl
  • text-2xl
  • text-8xl

Font Weights:

  • font-thin
  • font-extralight
  • font-light
  • font-normal
  • font-semibold
  • font-bold

Text Styles and Transformations:

  • Styles: italic, not-italic
  • Transformations: uppercase, lowercase, capitalize
  • Decorations: underline, line-through, no-underline
  • Line Height: leading-tight, leading-normal, leading-loose
  • Letter Spacing: tracking-tight, tracking-normal, tracking-wide

Borders

Border Width:

Border Radius:

  • rounded-sm, rounded-md, rounded-lg, rounded-xl

Background Colors:

Layouts

Width and Height:

  • w-1/2, w-full, h-32, h-screen

Padding and Margin:

  • Padding: p-4, py-2, px-2
  • Margin: mx-auto, mt-4, mb-2

Display and Position:

  • Display: block, inline-block, inline, table

Flexbox

Flex Containers:

  • flex: Defines a flex container. Enables flex context for its children.
  • flex-row: Places children in a horizontal row.
  • flex-col: Places children in a vertical column.

Tables

Tailwind CSS provides utility classes to style tables easily and effectively. Here's an overview of the key classes and their usage:

Table Layout:

  • table
  • table-auto: Automatically determines the column widths.
  • table-fixed: Fixes the table layout, useful for fixed-width columns.

Table Borders:

  • border: Adds a border to the entire table.
  • border-collapse: Collapses the table borders into a single border.
  • border-separate: Separates the table borders.

Table Rows and Cells:

  • table-row
  • table-cell
  • table-header-group: Defines a group of header rows.
  • table-footer-group: Defines a group of footer rows.
  • table-row-group: Defines a group of body rows.

Table Styling:

  • text-left, text-center, text-right: Aligns text within cells.
